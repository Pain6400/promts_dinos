# Experto en Prompts para CapCut

Esta habilidad te permite generar prompts y guiones diseñados específicamente para las herramientas de IA de CapCut (Magic Video, Script-to-Video). Tu objetivo es maximizar la calidad visual y la coherencia con las tendencias de edición de video corto (TikTok/Reels/Shorts).

## Cuándo usar esta habilidad
- Cuando el usuario quiera crear un video usando la función de "Texto a Video" de CapCut.
- Cuando el usuario pida estilos visuales específicos para CapCut (ej. "Cyberpunk", "Anime", "3D Cartoon").
- Para estructurar guiones que la seccion de "Script-to-Video" pueda interpretar mejor (división por escenas).

## Flujo de Trabajo

### 1. Definición del Estilo y Formato
Antes de generar el prompt, identifica:
- **Relación de Aspecto**: ¿9:16 (TikToK/Reels) o 16:9 (YouTube)?
- **Estilo Visual**: ¿Realista, Anime, Pintura al óleo, 3D Render?
- **Duración Estimada**: Corto (<30s) o Largo (>1m).

### 2. Estructura del Prompt para CapCut
La IA de CapCut responde mejor a descripciones concisas y visuales.

#### Para Generación de Video (Texto a Video)
Usa esta fórmula:
`[Sujeto principal] + [Acción] + [Entorno] + [Estilo Artístico] + [Iluminación/Atmósfera]`

*   *Ejemplo*: "Un T-Rex rugiendo en una ciudad futurista de neón, estilo cyberpunk 3D, iluminación cinematográfica, 4k."

#### Para Guion a Video (Script-to-Video)
CapCut divide el texto en escenas. Estructura el output así:

```text
# Título del Video
[Escena 1]
Narración: (Texto que dirá la voz en off)
Visual: (Descripción de lo que se ve para que la IA busque el stock o genere la imagen)

[Escena 2]
Narración: ...
Visual: ...
```

### 3. Palabras Clave "Mágicas" para CapCut
Incluye estos términos si buscas alta calidad:
- "Trending on TikTok"
- "High quality", "4k", "Ultra detail"
- "Cinematic lighting", "Smooth motion"
- Styles: "Anime", "Cyberpunk", "Oil painting", "Surrealism"
